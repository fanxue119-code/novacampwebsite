# Nova Camp 报名系统 - 页面导航文档

## 📱 项目概述

移动端H5报名系统，共17个页面，支持课程选择、报名表单、支付、拼团、课程管理等完整流程。

---

## 🔗 完整跳转关系图

### 1️⃣ 首页 (landing)

**出站跳转：**
- 点击右上角购物车图标 → `store`
- 点击"立即报名"按钮 → `course`
- 点击"了解更多"按钮 → `schedule`
- 点击"本期阵容"中的老师 → `teacher`
- 点击底部CTA按钮 → `course`

**导航按钮：** 仅购物车图标

## 🎯 核心用户路径

### 路径1：报名流程
```
首页 → 购买课程 → 报名表单 → 支付选择 → 完成
```

### 路径2：拼团流程
```
首页 → 购买课程（选拼团） → 拼团页面 → 邀请好友 → 支付
```

### 路径3：查看课程
```
首页 → 我的课程 → 我的课表 / 拼团详情
```

### 路径4：了解活动
```
首页 → Camp课表 → 比赛/After Party详情 → 报名/购票
```
